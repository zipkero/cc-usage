# Local Analytics Widgets

## 개요
Claude Code status line에서 현재 세션의 경과 시간, 토큰 소비 속도, 캐시 활용도, 토큰 구성, 한도 도달 예상, 시간당 비용 예측을 함께 확인할 수 있도록 로컬 분석 위젯 범위를 정의한다. 이 범위는 외부 CLI 연동이나 transcript 파싱 없이 stdin과 로컬 세션 상태만으로 관찰 가능한 분석 정보를 제공하는 데 집중한다.
